# InvoicePay

**InvoicePay** – The simple, powerful invoice tracker for freelancers and small businesses.

Send professional invoices, track payments, and get paid faster — all from your phone.

## Features

### Core (Free)
- Create unlimited invoices with customizable items
- Support for taxes, discounts, and line items
- Partial & full payment tracking
- Automatic overdue detection
- Beautiful, branded PDF generation
- Share via email, WhatsApp, or link
- Works offline
- Client management with notes & history

### Pro Features
- Unlimited clients & invoices
- Custom logo, company details, and branding on PDFs
- Auto-scheduled reminders (push + WhatsApp/email)
- Cloud backup & sync
- Advanced reports (income, overdue, tax summary)
- Ad-free experience
- Priority support

### Security & Privacy
- All data stored locally on device
- Optional encrypted cloud backup (Pro)
- No third-party analytics
- No data selling — ever

## Tech Stack
- Flutter (iOS & Android)
- Firebase (Auth, Firestore, Storage)
- RevenueCat (In-app purchases)
- flutter_local_notifications + workmanager (Reliable reminders)
- pdf package (Invoice generation)
